js-css-minify
=============

[Node.js 在线压缩 JS/CSS 代码](http://minify.coding.io/)

## Preview

![JS-CSS-Minify](./js-css-minify.png "JS-CSS-Minify")

## Install

```
git clone git@github.com:mailzwj/js-css-minify.git minify
cd minify
npm install
```

## Run

```
npm start
```
然后，浏览器输入`http://localhost:3000`，即可访问
